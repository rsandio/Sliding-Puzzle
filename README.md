# Sliding Puzzle
Version 1.1.0

Simple sliding puzzle game made using JavaFX.

## Features
* Ability to choose your own background image
* Four difficulty settings

## How to play
Move the tiles to recreate the image by clicking on tiles adjacent to the blank tile.
